# scoop-fileuni

Official Scoop bucket for FileUni CLI and GUI packages.

This repository is generated by the FileUni release workflow.
Do not edit generated manifest files manually.

## What You Get

- Manifest: `fileuni` (CLI)
- Manifest: `fileuni-gui` (GUI)
- Supported platforms:
  - `fileuni`: Windows x64, Windows x86
  - `fileuni-gui`: Windows x64

## Install

```powershell
scoop bucket add fileuni https://github.com/FileUni/scoop-fileuni
scoop install fileuni/fileuni
scoop install fileuni/fileuni-gui
```

## Upgrade

```powershell
scoop update
scoop update fileuni
scoop update fileuni-gui
```

## Verify

```powershell
fileuni --help
fileuni-gui
```

## Uninstall

```powershell
scoop uninstall fileuni
scoop uninstall fileuni-gui
```

## Files

- CLI manifest: `bucket/fileuni.json`
- GUI manifest: `bucket/fileuni-gui.json`
- Install guide: `INSTALL.md`
